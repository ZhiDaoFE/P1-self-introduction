之道前端

学前端，来之道 —— 陪伴式自学前端圈子

之道前端是什么？

之道前端是国内专业的 自学前端交流圈，是最适合前端开发 持续提升技术能力 的学习场域。
之道前端的目标是：给你最实用的自学指引，送你最科学的原创项目，陪你在自学成长的路上一直走下去。

之道前端有什么？

你只需要跟着这个路线一路学习就可以了，之道前端包你学会前端。

二、原创项目

之道的前端自学路线中，包含了大量的原创项目，这些配套的原创项目适配你当前的学习进展和技术水平，
利用已经学到的知识去动手写代码，完成 你的作品，获得兴趣和成就感，激励你继续学习，形成 良性循环。

之道前端不会一上来就给你一个 XXX 外卖系统，有 C 端和 B 端，
之道前端不会一上来就给你一个包含前端后端的完整全栈系统，
之道前端不会一上来就给你一个高大上的，堆砌各种框架、中间件的复杂系统，

之道前端的原创项目不是最高大上的，但一定是 最适合你的。

当然，之道前端也会有综合性的大型项目，而且是贴近大厂真实场景的原创项目，而不是自己 YY 的，强硬堆砌各种框架、各种中间件的系统。
这是要到很后面才会出现的原创项目了，一定不会让你失望的，是完全可以放进你简历里的项目。
你也不用担心学不会，只要你跟着之道的前端自学路线一路学练下去，完成这些综合项目就是水到渠成的事。

三、陪伴式自学圈子

一个人可以走得很快，但一群人才能走得更远

之道前端帮你寻找自学前端的小伙伴，连接更多资源和人脉，让你在自学前端路上 不孤单。
之道前端帮你寻找前端大牛，连接更多经验和导师，让你在自学前端路上 不迷茫。

答疑解惑

在之道前端知识星球中，如果你碰到任何疑问，你可以阅读星球专栏、查阅其他小伙伴的提问，寻找类似的提问和解答。
如果没找到类似的提问，你也可以直接向导师提问。

学习氛围

在自学前端的过程中，还有一个绊脚石就是 难以坚持。
之道前端会 为你营造学习氛围，推着你前进，拒绝躺平。
在之道前端知识星球中，会有各种各样的分享、打卡活动和作业活动，参与这些服务，会让你在自学过程中，形成思考和总结的习惯。
只看是学不到东西的，你要参与、动手、实践、思考，才能真正地成长。

四、专属直播，经验分享

之道前端会不定期举办 圈友专属 的直播，内容包括但不限于手把手带做项目、前端开发经验分享、前端知识点精讲、水友会专场直播等，内容丰富，形式多样。
加入之道前端，就可以免费参加所有的专属直播，如果你的时间不匹配，也可以随时观看超清回放。

我适合加入么？

如果你是以下情况，之道前端非常欢迎你：

  • 零基础想要自学前端，但却不知如何开始
  • 正在自学前端，但缺少目标、感到迷茫、甚至自我怀疑
  • 正在自学前端，但很难坚持，想要找搭子
  • 正在自学前端，但难以真正学进去，经常忘记之前学过的内容，进展超慢
  • 想要找自学前端的搭子
  • 想和大厂高工近距离交流，学习经验，听听分享
  • 在公司，天天做业务，千篇一律，毫无成长
  • 在公司，没有导师，leader 不指导，不知道该如何成长
  • 在公司，都是做 admin 页面，或者只负责很小的模块，成长空间极为有限
  • 在公司，觉得再这么待下去，人就要废了，但是短时间内又难以找到合适的 offer
  • 只要你想自学前端，并且不想躺平

如果你是以下情况，不适合 加入：

  • 不打算做程序员
  • 不打算做前端
  • 有能力可以自学前端，有方向，有规划，有方法，能坚持
  • 参加工作 >= 3 年，有一定的技术水平和成熟的经验
  • 目前业务领域涉及面广，前端团队成熟并专业，有很多项目实践机会
  • 能轻松找到 offer

如果你想要自学前端，提升自己的能力，
你要做的只需要加入之道前端，然后专心学练，
剩下的，之道前端都会为你准备好。
